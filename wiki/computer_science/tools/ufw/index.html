UFW

description: Firewall

lang: ENG

Description

ufw is a firewall on top of iptableQ

Commands

Status

ufw status [verbose]

Enabled

ufw enable

Allow port from anywhere (IPv4-6)

ufw allow <PORT>

Deny incoming by default

ufw default deny incoming
