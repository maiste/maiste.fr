Registars

  • Gandi: french registar - smaller size company - really efficient
  • OVH: french registar - also provide bare metals servers

Cloud Providers

  • Equinix: bare metal server for quick executions
  • Digital Ocean: cheap VPS if you need a small size
  • Scaleway: European base VPS - still expensive

Learning

  • Learn X in Y: summaries to learn tools and languages
  • Effective shell: book about using the shell in an efficient way

Version Control manager

  • Gitlab: open source manager - Pipeline for CI
  • Github: Microsoft manager - no open source instance - Github Actions for CI

Web Provides

  • Netifly: web provider - free for small static website or small build one
  • Disposable root servers: free root servers in ssh for Hackers

  • End of Life: information about End Of Life of various projects
  • Pingdom: tests the speed of your website
