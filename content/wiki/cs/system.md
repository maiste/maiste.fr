## Update Fireware

### Commands

* Check detected devices:
```
fwupdmgr get-devices
```

* Refresh latest metadata:
```
 fwupdmgr refresh
```
* List available update:
```
fwupdmgr get-updates
```
* Install update:
```
fwupdmgr update
```
